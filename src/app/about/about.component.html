About the application
Daily Tasks
  • Sometimes because of the busy schedule, we miss some of the tasks or forget daily.
  • Daily Tasks helps you perform tasks without fail in your busy schedule.
  • Users can create daily tasks which are to be performed daily.
  • The tasks under daily tasks gets refreshed daily at 12 am.
  • Unless deletion of the task, the tasks under daily tasks gets refreshed.
  • User can Delete his/her tasks or Add tasks
  • Examples -
    1. Daily time sheet filling.
    2. Consuming milk/water.
    3. Doing exercises.
    4. Taking medicines, etc.
General Tasks
  • General tasks, to help you remember the tasks which needs to be done./li>
  • Unlike Daily tasks, General tasks will not refresh.
  • Once checked it is considered done or get deleted..
  • Unless deletion of the task, the tasks cannot be removed.
  • User can Delete his/her tasks or Add tasks
  • Examples -
    1. Vacation lists.
    2. Grocery lists.
    3. Things to be bought.
    4. Sending important mails etc.
